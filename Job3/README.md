## Description  
   This project explores various optimization strategies for parallel reduction using CUDA.  
   To run:  
     **mkdir build bin data**  
     **cd scripts**
     **g++ genMat.cpp -o genMat**
     **cd ..**
     **./scripts/genMat**
     **make**  
     **make run**  
     **make clean**
